# ☁️ Cloud Dataprep Overview

**Cloud Dataprep** is an intelligent, serverless data service for **exploring, cleaning, and preparing data** for analysis, reporting, and machine learning.

---

## 1️⃣ Key Features

- **Serverless & scalable:** No infrastructure to deploy or manage  
- **Visual interface:** Explore and clean data without writing code  
- **Smart suggestions:** Automatically predicts next transformations based on UI inputs  
- **Automatic profiling:** Detects schemas, data types, anomalies, and possible joins  
- **Focus on analysis:** Reduces time spent on data preparation  

> ⚡ Cloud Dataprep is built on **Trifacta Wrangler**, integrated with Google Cloud for a seamless experience without software installation or operational overhead.

---

## 2️⃣ Architecture & Workflow

1. **Data sources:** Ingest raw data from:  
   - `BigQuery`  
   - `Cloud Storage`  
   - File uploads  
2. **Data preparation:** Clean, transform, and enrich data using Cloud Dataprep  
3. **Pipeline integration:** Feed refined data into transformation pipelines like **Cloud Dataflow**  
4. **Export for analysis:** Send cleaned data to:  
   - `BigQuery` for analytics  
   - `Cloud Storage` for storage or machine learning  

---

## 3️⃣ Benefits

- No upfront licensing or software installation  
- Fully managed and scales on-demand  
- Reduces manual coding and operational overhead  
- Streamlines data workflows from raw ingestion to analytics or ML
